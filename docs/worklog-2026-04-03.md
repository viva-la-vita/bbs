## 2. 部署文档修正

**问题：** 容器内是 BusyBox，`find` 不支持 `\( \)` 分组语法，原文档命令报错。

**修复后的删除命令：**
```bash
docker exec bbs-flarum-1 sh -c 'cd /var/www/flarum/public/assets && rm -f *.js *.css *.map *.json'
docker exec bbs-flarum-1 rm -rf /var/www/flarum/public/assets/extensions
docker exec bbs-flarum-1 rm -rf /var/www/flarum/public/assets/fonts
docker exec bbs-flarum-1 php flarum assets:publish
# cache:clear 需执行两次，防止偶发白屏
docker exec bbs-flarum-1 php flarum cache:clear
docker exec bbs-flarum-1 php flarum cache:clear
```

## 4. bbs-antispam：自动封号扩展

**功能：** 注册不足90天的用户，被3个不同用户（各自注册满7天）举报后，自动暂停账号并隐藏其所有帖子和主题。

**触发条件（同时满足）：**
1. 被举报账号注册 < 90 天
2. 累计 ≥ 3 个不同用户举报（跨帖累计）
3. 举报人自身注册 ≥ 7 天

**触发动作：**
1. 隐藏该用户所有帖子和主题
2. 暂停账号（10年，相当于永久）
3. 在 Flags 队列创建 `autoSuspend` 通知（仅管理员可见）

**解封命令：**
```bash
docker exec bbs-flarum-1 php flarum antispam:unsuspend <用户名>
```
解封同时恢复该用户所有帖子和主题可见性。

**新增文件：**
- `flarum/www/packages/bbs-antispam/`（全新扩展）

**同步操作：** 关闭 Flagify 插件（已被新方案替代）。
